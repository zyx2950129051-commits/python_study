# 3. 运行 Python 程序的几种方式总结

运行 Python 程序，有常见的以下三种方式：

第一种方式：命令行（终端）模式

第二种方式：脚本模式

第三种方式：集成开发环境（IDE）模式

备注：第三种方式，其实是第二种方式的图形化操作，本质上算是一种模式。

## 3.1. 命令行模式

同时按下 Win 键和 R ，随后输入cmd ，打开终端（命令行）。

在终端（命令行）中输入python，进入 Python 交互模式。

输入print(100)，按下回车，控制台会打印：100。
